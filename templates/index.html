💪 2. Masa magra (Lean Body Mass)
Fórmula Masa grasa (kg) = Peso × (%GC / 100) Masa magra (kg) = Peso − Masa grasa
Paso 1 · Masa grasa {{ p.mm_peso }} × ({{ p.navy_grasa }} / 100) = {{ p.mm_masa_grasa }} kg
Paso 2 · Masa magra {{ p.mm_peso }} − {{ p.mm_masa_grasa }} = {{ p.mm_masa_magra }} kg Masa magra = {{ p.mm_masa_magra }} kg

La masa magra incluye músculos, huesos, agua corporal y órganos, excluyendo la grasa. Es el indicador clave para seguimiento de rendimiento deportivo.

Fórmula IMC = Peso (kg) / Altura (m)²

Paso 2 · Calcular IMC {{ p.imc_peso }} / {{ p.imc_altura_m }}² = {{ p.imc_peso }} / {{ p.imc_altura_m_cuadrado }} = {{ p.imc_valor }} kg/m² IMC = {{ p.imc_valor }} kg/m² → Categoría OMS: {{ p.imc_categoria }}

Clasificación OMS: <18.5 Bajo peso · 18.5–24.9 Peso normal · 25–29.9 Sobrepeso · 30–34.9 Obesidad I · 35–39.9 Obesidad II · ≥40 Obesidad III. El IMC no distingue entre masa grasa y masa magra.

🏋️ 4. Índice de Masa Libre de Grasa (FFMI)
Fórmula FFMI = Masa magra (kg) / Altura (m)² FFMI normalizado = FFMI + 6.1 × (1.80 − Altura m)
Paso 1 · FFMI bruto {{ p.ffmi_masa_magra }} / {{ p.ffmi_altura_m }}² = {{ p.ffmi_masa_magra }} / {{ p.ffmi_altura_m_cuadrado }} = {{ p.ffmi_valor }} kg/m²
Paso 2 · FFMI normalizado a 1.80 m {{ p.ffmi_valor }} + 6.1 × (1.80 − {{ p.ffmi_altura_m }}) = {{ p.ffmi_norm_valor }} kg/m² FFMI = {{ p.ffmi_valor }} kg/m² · FFMI norm. = {{ p.ffmi_norm_valor }} kg/m²

El FFMI normalizado permite comparar individuos de distintas estaturas. Referencia natural: ♂ <25 kg/m² · ♀ <22 kg/m². Valores superiores pueden indicar uso de sustancias.
