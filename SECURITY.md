# Política de Seguridad

| Versión | Soporte          |
|---------|------------------|
| 0.1.x   | ✅ Activo         |

## Reportar una vulnerabilidad

Si descubres una vulnerabilidad de seguridad en este proyecto, **no la reportes como issue público**. En su lugar:

1. Envía un correo electrónico al mantenedor del proyecto describiendo la vulnerabilidad (puedes encontrar el contacto en el perfil de GitHub [@juanfranciscofernandezherreros](https://github.com/juanfranciscofernandezherreros)).
2. Incluye la siguiente información:
   - Descripción del problema y su impacto potencial.
   - Pasos para reproducirlo (PoC si es posible).
   - Versión afectada.
3. Recibirás una confirmación en un plazo de **72 horas** y trabajaremos contigo para resolver la vulnerabilidad antes de divulgarla públicamente.

## Divulgación responsable

Seguimos una política de **divulgación coordinada**: una vez corregida la vulnerabilidad se publicará un aviso de seguridad en el repositorio agradeciendo la contribución (a menos que el reportador prefiera permanecer anónimo).

Gracias por ayudar a mantener este proyecto seguro. 🔐
